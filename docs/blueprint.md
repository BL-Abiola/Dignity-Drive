# **App Name**: Dignity Drive

## Core Features:

- Direct Bank Transfer Information: Display bank account details prominently for direct donations.
- Copy to Clipboard: Implement a 'Copy to Clipboard' function for the bank account number using JavaScript.
- Mission Highlights: Showcase the impact of donations with images of past outreach events.
- Essential Dignity Items: List the key items provided, like cash, clothes, food, and toiletries, to clearly communicate what donations achieve.
- Contact Information: Provide a contact phone number for inquiries.

## Style Guidelines:

- Primary background: Clean off-white (#FAFAFA) for a fresh look.
- Hope/Trust color: Vibrant gold/yellow (#FFD700) to inspire confidence.
- Urgency/Accent color: Deep firebrick red (#B22222) to highlight calls to action and represent the mission.
- Font: 'Poppins', a geometric sans-serif font, via Google Fonts, for a sharp and modern feel.
- Rounded corners (rounded-xl) and shadow effects (shadow-lg) on key information cards to build trust.
- Visually distinct donation block using background #FFD700 and border-4 border-yellow-700 to attract the user's attention.
- Use relevant emojis/icons to illustrate the 4 key items provided: Cash, Clothes, Foodstuffs, Toiletries.
